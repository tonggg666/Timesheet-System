# ⏱️ 专业工时记录系统 (Timesheet System) V4.1

这是一个基于纯 HTML、CSS 和 JavaScript 构建的简易工时记录应用。它利用浏览器本地存储 (LocalStorage) 来安全地保存员工档案和每日工时记录，无需数据库或服务器。

## 🚀 主要功能

* **员工档案建档与管理:** 录入员工的 ID、姓名、公司和工种。
* **模态框管理:** 通过弹窗 (Modal) 清晰地管理（编辑/删除）员工档案，避免页面过长。
* **批量工时录入:** 选择日期，为所有已建档员工批量录入当日工时。
* **数据安全:** 所有数据本地保存在您的浏览器中。
* **Excel 导出:** 一键生成包含所有员工和每日工时数据的月度 Excel 报表。

## 🛠️ 使用方法 (本地运行)

1.  下载或克隆本仓库。
2.  在文件管理器中找到 `timesheet.html` 文件。
3.  双击打开该文件，它将在您的默认浏览器中运行。

> **注意：** 首次使用时，如果无法下载 Excel 文件，请确保您的电脑连接到互联网，因为系统依赖于 SheetJS CDN 库来处理 Excel 导出。
